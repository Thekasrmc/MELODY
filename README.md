# 📚 MELODY Bot - คู่มือการใช้งาน

> **🎵 Discord Music Bot สำหรับเล่นเพลงบน Discord ใช้งานง่าย ควบคุมได้ผ่านคำสั่งและปุ่มกด**
> 
> MELODY Bot คือบอทสำหรับเล่นเพลงใน Discord ที่สามารถ:
> เล่นเพลงจาก YouTube / Spotify / SoundCloud
> ควบคุมเพลงด้วยคำสั่งและปุ่มกด
> จัดคิวเพลงอัตโนมัติ
> แสดงสถานะเพลงแบบเรียลไทม์

## 📌 วิธีใช้งานเบื้องต้น
1. เข้าห้องเสียง (Voice Channel)
2. พิมพ์คำสั่ง /play
> บอทจะเข้ามาเล่นเพลงให้ทันที

## 🎵 คำสั่งทั้งหมด
### ▶️ /play <ชื่อเพลงหรือ URL>

**ใช้สำหรับ:**

```เล่นเพลง```
```เพิ่มเพลงเข้าคิว```

**ตัวอย่าง**
```
/play เพลงฮิต
/play https://youtube.com/...
/play เพลงสากล
```
**การทำงาน**

> ```ถ้ายังไม่มีเพลง → เล่นทันที``` ```ถ้ามีเพลงอยู่แล้ว → เพิ่มเข้าคิว```
### ⏭️ /skip

**ข้ามเพลงที่กำลังเล่นอยู่**

> ```เพลงถัดไปจะเล่นทันที```
```ถ้าคิวหมด → บอทจะหยุด```

### ⏹️ /stop

**หยุดการเล่นทั้งหมด**

```ล้างคิวเพลง```
```บอทออกจากห้องเสียง```
### ⏸️ /pause

**หยุดเพลงชั่วคราว**

### ▶️ /resume

**เล่นเพลงต่อจากที่หยุดไว้**

### 📋 /queue

**ดูรายการเพลงทั้งหมดในคิว**

```แสดงลำดับเพลง```
```ดูได้หลายหน้า ถ้ามีเพลงเยอะ```
### 🎶 /nowplaying

**ดูเพลงที่กำลังเล่นอยู่**

**จะแสดง:**

```ชื่อเพลง```
```เวลาที่เล่นไปแล้ว```
```แถบความคืบหน้า```
```ผู้ที่ขอเพลง```

### 🔀 /shuffle

**สุ่มลำดับเพลงในคิว**

### 🔁 /loop <โหมด>

**โหมดการเล่นซ้ำ:**

| สิ่งที่สั่ง | ผลลัพธ์ |
| :--- | :--- |
| **OFF** |ปิดการเล่นซ้ำ |
| **TRACK** | เล่นเพลงเดิมซ้ำ |
| **QUEUE** | เล่นทั้งคิวซ้ำ |

**ตัวอย่าง**
```
/loop OFF
/loop TRACK
/loop QUEUE
```
### ❌ /remove <ลำดับ>

**ลบเพลงออกจากคิว**

**ตัวอย่าง**
```
/remove 1
/remove 3
```
### 🧹 /clear

**ลบเพลงทั้งหมดในคิว (ไม่รวมเพลงที่กำลังเล่น)**

### 🎚️ /volume <ระดับเสียง>

**ปรับระดับเสียงเพลง**

**ตัวอย่าง**
```
/volume 50
/volume 100
/volume 150
```
```100 = ปกติ```
```0 = เงียบ```
```สูงสุดที่ 200```
## 🎮 ปุ่มควบคุม (กดได้ในข้อความเพลง)

**เมื่อใช้ /nowplaying หรือ ใช้ /play เเล้วจะมีปุ่มให้กด:**

**⏸️ Pause / Resume**
```หยุด / เล่นต่อ```

**⏭️ Skip**
```ข้ามเพลง```

**⏹️ Stop**
```หยุดทั้งหมดและออกจากห้อง```

**🔁 Loop**
```เปลี่ยนโหมดเล่นซ้ำ```

**📋 Queue**
```ดูคิวเพลงแบบเร็ว```

## ⚠️ ข้อจำกัดการใช้งาน
```ต้องอยู่ในห้องเสียงก่อนใช้คำสั่ง```

```จำกัดจำนวนเพลงในคิว```
## 💡 วิธีใช้งานแนะนำ
### 🎧 เปิดเพลงง่าย ๆ
```/play ชื่อเพลง```
### 🎶 เปิดเพลงยาวต่อเนื่อง
```/loop QUEUE```
### 🔥 เปิดเพลงต่อเนื่องแบบไม่หยุด
```/play เพลง 1,2,3,4,5```
```จากนั้นเพิ่มเพลงไปเรื่อย ๆ```
## 🐛 ปัญหาที่พบบ่อย
### ❓ บอทไม่เล่นเพลง

```เช็คว่าคุณอยู่ในห้องเสียงหรือยัง```

```ลองใช้ /play ใหม่อีกครั้ง```

### ❓ บอทออกจากห้องเอง

```ไม่มีคนอยู่ในห้องเสียง```

```หรือไม่มีเพลงในคิวแล้ว```

### ❓ เพลงไม่ขึ้นคิว

```ลองใช้ชื่อเพลงให้ชัดเจนขึ้น```

```หรือใช้ลิงก์ YouTube แทน```
